A collection of handy utilities for js/ts programing

## Quik Start

```bash
npm install ytils
```

## Usage

CommonJs:

```js
const { isEq } = require("ytils");
console.log(isEq({ name: "xavier" }, { name: "xavier" })); // => true
```

ES Module:

```js
import { isEq } from "ytils";
console.log(isEq({ name: "xavier" }, { name: "xavier" })); // => true
```
